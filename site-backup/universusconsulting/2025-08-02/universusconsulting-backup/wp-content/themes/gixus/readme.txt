== Installation ==
1. Upload the `gixus` folder to the `/wp-content/themes/` directory or install the theme through the Themes option in WordPress.
2. Activate the theme through the 'Appearance' menu in WordPress.
3. Navigate to Pages > 'Edit with Elementor Page Builder' to start customizing the pages.

For detailed instructions, refer to the [Gixus Theme Documentation](https://ridhwan.gitbook.io/wordpressriver-themes-help-center).

== Frequently Asked Questions ==

= How do I import demo content? =
To import demo content, install and activate the One Click Demo Import plugin, then go to 'Appearance > Import Demo Data' and follow the instructions.
